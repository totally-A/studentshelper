Цикл While у мові С

Оператор циклу while або цикл while - цикл , що повторює одну і ту саму дію, поки умова продовження циклу while залишається істинною. 

// Форма запису циклу while
while ( / * умова продовження циклу while * / )
	{ 
		/ * Блок операторів * / ;
		/ * Управління умовою * /;
	}

Умова продовження циклу має бути істинно true, як тільки умова стала хибною, виконується вихід з циклу. Також як і в умовних операторах вибору, фігурні скобочки можуть опускатися в тому випадку, якщо тіло циклу - це один оператор. Але як правило в циклі виконується кілька операторів, так як крім виконання корисної дії необхідно робити умову циклу while хибною, інакше цикл буде нескінченним, а це, в свою чергу, призведе до зависання програми.

Розглянемо застосування циклу while на прикладі руху автомобіля. На псевдокоді, то це буде так:

Поки швидкість руху автомобіля менше 60 км/год, продовжувати збільшувати швидкість.

Справжньою умовою циклу, в даному випадку, є швидкість автомобіля менше 60 км / год, а хибністю - швидкість автомобіля більше або дорівнює 60 км / год. Повторення циклу триватиме до тих пір поки швидкість автомобіля не стане більше або дорівнює 60 км / год, після настання цього моменту умова циклу стане хибною, і програма вийде з циклу. Розглянемо фрагмент коду на C з циклом while вирішальним поставлену задачу. 

int speed = 5; // Початкова швидкість автомобіля
while (speed < 60) // заголовок циклу while
speed += 10; // Тіло циклу

Гарний стиль програмування

  • Ініціалізуйте лічильники та загальні суми.
  • При виконанні ділення на вираз, значення якого може бути нуль, явно робіть перевірку цього випадку та оброблюйте його належним чином(наприклад, виводьте повідомлення про помилку), так не допускайте виникнення фатальної помилки.
  • У циклі, який керується контрольним значенням, підказка для запиту введення даних повинні явно нагадувати користувачу, чому дорівнює це значення.

Приклад програми

Для закріплення матеріалу по циклу while розробимо всім відому гру «Відгадай число». ЇЇ реалізовувало дуже багато програмістів, тому вона не є новинкою. На псевдокоді це буде так:

// Поки загадане число не вгадано, продовжувати вгадувати його

Загадувати число буде комп'ютер, а ми - відгадувати. Використовуючи генератор випадкових чисел, комп'ютер загадає число від 1 до 10, включно, поки ми не відгадаємо число вихід з циклу не відбудеться. 

#include< iostream >
#include< ctime >

int main()
{
	system("color F0");
	srand(time(0));
	int unknown_number = 1 + rand() % 10;              //guessing number
	int enter_number;                                  //value to store the entered number
	puts("Enter unknown number from 1 to 10:");        //begin to guess
	scanf("%d", &enter_number);
	while (enter_number != unknown_number)
	{
		puts("Enter unknown number from 1 to 10:");
		scanf("%d", &enter_number);                    //continue to guess
	}
	puts("You win!!!\n");
	system("pause");
}
