Документація

Повне найменування

Електронний довідник з гарного стилю програмування та типовими помилками на С/С++

Коротка характеристика області застосування програми:

Даний продукт може застосовуватися для будь-якого користувача, який бажає гарно та без помилок навчитися писати код. Проект поки орієнтовано на кафедру АСОІУ НТУУ «КПІ» для перевірки знань студентів та отримання нових навичок. Також схожі тести можна використовувати в школах, університетах, автошколах, на виробництві для тестування знань.

Призначення розробки

Даний програмний продукт призначений для використання в навчальних цілях, з метою ознайомлення студентів з гарним стилем програмування, покращенням якості коду, який пишуть розробники. Також проект створений для загального удосконалення базових знань та вже набутих умінь у сфері програмування. За допомогою створеного сайту навчання проходить більш ефективніше. За допомогою тестів перевіряються знання студентів.

Технічні вимоги

Алгоритмічна частина
Проект розроблений засобами WEB-розробки - HTML, CSS. Також є програми розроблені мовою С. Для створення цим програм потрібні навички для роботи з бінарними файлами, структурами та масивами структур.

Програмна частина
Реалізує ідеї WEB-розробки, програми заповнення бінарного файлу та тестування написані мовою С.

Вимоги до надійності
Програмі тестування має бути надано доступ до файлу з вхідними даними. Проте вхідний файл не має бути показаний користувачеві, адже там містяться відповіді на питання.

Умови експлуатації
Управління програмою тестування відбувається при введенні з клавіатури номера операції, що вказана в меню і натиском клавіші ENTER. На сайті присутня навігація, яка зрозуміла кожному користувачеві.

Вимоги до складу і параметрів технічних засобів
Процесор – Intel Pentium 1,66 ГГц;
Об’єм оперативної пам’яті – 512 Мб;
Об’єм жорсткого диску – 2 Гб;
Операційна система: MS Windows XP та вищі версії;
Середовище розробки: MS Visual Studio 2010, наявність Microsoft .NET Framework.
Наявність клавіатури.

Вимоги до інформаційної і програмної сумісності
Програмний код має бути написано виключно мовою програмування Visual Studio ANSI C++. Для написання меню користувача допускається використання HTML, CSS та JavaScript.

Вимоги до транспортування та зберігання
Запобігати потраплянню сторонніх речовин всередину упаковки, зберігати за кімнатної температури. Термін зберігання даних на диску – до 10 років.
